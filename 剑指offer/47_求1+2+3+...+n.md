# 题目描述
求1+2+3+...+n，要求不能使用乘除法、for、while、if、else、switch、case等关键字及条件判断语句（A?B:C）。
# 解析
使用递归`f(n)=f(n-1)+n`
# 代码
```java
public class Solution {
    public int Sum_Solution(int n) {
        //求f(n)
        int result;
        Solution sl = new Solution();
        
        if(n==1)
            result = 1;
        else {            
            result = sl.Sum_Solution(n-1)+n;
        }
        return result;
    }
}
```
